# Bahasa Indonesia kos-kosan glossary

Quick reference for parsing raw Mamikos / Rumah123 / OLX listings without mistranslating.

## Listing types

| Bahasa | English | Notes |
| --- | --- | --- |
| `kos` / `kost` / `kos-kosan` | Boarding-house room | Single rented room, often in a multi-room building. Owner usually lives on site. |
| `kontrakan` | Rented house | Whole house — out of scope for this skill. |
| `rumah` | House | Out of scope. |
| `apartemen` | Apartment | Out of scope. |

## Money & payment

| Bahasa | English | Example |
| --- | --- | --- |
| `Rp` | Rupiah | `Rp 1.500.000` = 1,500,000 IDR |
| `jt` / `juta` | Million | `1,5jt` = 1,500,000 |
| `M` / `miliar` | Billion | `1,5M` = 1,500,000,000 (NOT million — common LLM trap) |
| `rb` / `ribu` | Thousand | `500rb` = 500,000 |
| `/bulan` | Per month | |
| `/tahun` | Per year (often paid upfront) | |
| `/minggu` | Per week | |
| `/hari` | Per day | |
| `bayar di muka` | Paid upfront | |
| `DP` | Down payment / deposit | |
| `nego` / `nego tipis` | Negotiable / slightly negotiable | |
| `include listrik` | Electricity bundled | |
| `exclude listrik` | Electricity metered separately | |

### Number-format trap

Indonesian uses `.` as thousands separator and `,` as decimal:
- `1.500.000` = one million five hundred thousand
- `1,5` = one point five
- `Rp 1,5jt` = `Rp 1.500.000` = 1,500,000 IDR
