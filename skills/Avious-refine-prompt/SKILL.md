# Refine Prompt

Turn a rough draft prompt into a high-quality one — you improve the prompt as an artifact, you do not execute its task. **Read [reference.md](reference.md) in this skill's directory for the input sources, principles to apply, the fixed output structure, and rules — then follow it.**
